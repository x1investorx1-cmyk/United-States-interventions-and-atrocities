# American Reckoning

**A censorship-resistant archive of 100 U.S. interventions, covert operations, and atrocities since 1945.**

---

## Purpose

Governments control information. Search results get filtered. Websites get blocked. Histories get rewritten.

This project exists because the documented record of American foreign policy — coups, massacres, proxy wars, torture programs, illegal bombings — is the kind of information that disappears in politically controlled environments. It has been compiled from declassified government documents, congressional investigations, independent journalism, and academic research. It belongs in the public domain, freely accessible, without gatekeepers.

By packaging everything into a single self-contained HTML file, this archive can be:

- Saved to a USB drive and shared hand to hand
- Emailed as an attachment
- Hosted on any static web service, free or paid
- Opened directly in a browser with no internet connection required
- Mirrored anywhere, instantly, at zero cost

No server. No database. No login. No tracking. Just a file and a browser.

---

## What's Inside

- **100 documented events** ranked by human cost — from Hiroshima (1945) to the Forever Wars (2001–2021)
- **Unique illustration** for each event, embedded directly in the file
- **Full descriptions** sourced from the historical record
- **President**, **country**, **death toll**, and **category** for every entry
- **Search** by event name, country, president, or keyword
- **Filter** by category: Military, CIA Ops, Nuclear, Drone, Economic, and by region

## How to Use

**Online (GitHub Pages):**
1. Fork or download this repository
2. Enable GitHub Pages under Settings → Pages
3. Share the link — it works in any browser, anywhere

**Offline:**
1. Download `index.html`
2. Double-click to open in any browser
3. Everything works — no internet required

**To share without internet:**
- Copy `index.html` to a USB drive, phone, or SD card
- The recipient opens it in any browser
- No installation, no app, no account needed

---

## Philosophy

> *"The most dangerous thing in an authoritarian system is a fact that can be verified."*

This archive does not require trust in any platform, server, or institution to remain accessible. Once downloaded, it cannot be taken down, filtered, or modified without the user's knowledge. That is the point.
